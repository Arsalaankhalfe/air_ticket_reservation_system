air_ticket_reservation_system


This project is a terminal-based administrative air ticket reservation system developed using Python and MySQL. It automates passenger registration, class-based ticket pricing, food ordering, luggage billing, and total fare calculation through a structured menu-driven interface.

The system demonstrates practical implementation of database connectivity, SQL queries, modular programming, and real-time cost computation using Python.


FEATURES

- Passenger registration with database storage

- Travel class selection and dynamic ticket price calculation

- Food ordering system with quantity-based billing

- Luggage weight-based charge calculation

- Integrated total cost computation

- Menu-driven terminal interface

HOW TO RUN

- pip install mysql-connector-python
- python main.py
- connect with sql
